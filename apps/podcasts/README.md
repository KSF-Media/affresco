# Podcasts

## DEV Setup
`yarn install`

## DEV run
Get the user id from the podcasts you want to include in this app from Soundcloud.

Add the ids in `.env` separated by comma, like  
`PODCAST_IDS=542583531,694513583,705599305,630339678`

Serve lambda funtions  
`yarn lambda serve`

Run dev server  
`yarn start`

## Build bundle
`yarn build`
